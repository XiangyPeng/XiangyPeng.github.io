## 📌 Project Overview
This work focuses on building an end-to-end edge–cloud cooperative sensor data ingestion system for real-time perception and monitoring. The system combines embedded inference on MCU hardware, gateway-based communication, and cloud-side data transmission to enable efficient radar data processing in resource-constrained environments.

## 🧠 My Contributions
- Designed an end-to-end sensor-edge–cloud data ingestion system, integrating PSoC6-based edge inference, Raspberry Pi gateway, and cloud-side monitoring for scalable deployment.
- Developed lightweight deep learning models in PyTorch for radar point cloud classification and segmentation, optimized for real-time perception tasks.
- Deployed TinyML models on MCU using TensorFlow Lite Micro, achieving efficient real-time inference under strict memory and compute constraints.
- Optimized model architecture and input pipelines to reduce latency and ensure stable performance on embedded hardware.
- Implemented multi-protocol communication using UART, Wi-Fi, BLE, and MQTT for reliable data transmission between edge devices, gateway, and cloud.
- Built a pipeline from sensor acquisition to cloud integration, enabling continuous monitoring, data flow, and system-level evaluation.
